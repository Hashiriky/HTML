Hierarquia de títulos

Cuidado com a Hierarquia

Os títulos vão de h1 até h6, sempre seguir a ordem de 1 a 6, NUNCA pular a ordem. Como exemplo, ir do h1 para o h4 é incorreto. Sempre seguir a ordem: h1, h2, h3, h4, h5 e h6. Não existe h0 e nem maior que h6.

Semântica

Html5 foca em Semântica, Significados, palavras.
''O HTML5 foca em usar tags que têm um significado claro, de modo que os navegadores e outros agentes de usuário possam interpretar o conteúdo das páginas da web de forma mais precisa.''

Destaques dos textos

Para colocar um texto em negrito use a tag (strong). A tag (b) poderá se tornar obsoleta no futuro. E para colocar um texto em itálico use a tag (em). A tag (i) também pode se tornar obsoleta.

As tags <b> e <i> são tags semânticas, o que significa que elas têm um significado específico. No entanto, o HTML5 recomenda o uso de tags semânticos mais específicas para formatar texto. Por exemplo, a tag <strong> é usada para enfatizar texto, enquanto a tag <em> é usada para indicar palavras ou frases que são importantes ou relevantes.
